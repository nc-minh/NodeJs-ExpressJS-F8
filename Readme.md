# NODE & EXPRESSJS

## Install

```
npm i express
```
```
npm i nodemon
```
```
npm i morgan
```
### Template engine (handlebars)
```
express handlebars
```
### Use sass
```
 npm install -g sass
```

## Run
```
cd blog
```
```
nodemon index.js
```
```
npm run watch
```
